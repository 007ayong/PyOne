# PyOne主题开发规范

## 主题放在哪里？

主题目录统一放在`PyOne/templates/theme/`下

## 必须有哪些页面？

以自带主题为例
```
theme
    └── material
        ├── 500.html
        ├── find.html
        ├── footer.html
        ├── header.html
        ├── head.html
        ├── index.html
        ├── layout.html
        ├── _macro.html
        ├── password.html
        ├── readme.html
        └── show
            ├── any.html
            ├── audio.html
            ├── code.html
            ├── image.html
            ├── video2.html
            └── video.htm
```
